2017년 7월 기준 홈택스 부가가치세 신고방법

  1. [홈택스 신고] 개인사업자 부가가치세 신고납부 : 일반과세자 기본정보 입력
  2. [홈택스 신고] 개인사업자 부가가치세 신고납부 : 일반과세자 신고내용 / 과세표준 및 매출세액 입력하기
  3. [홈택스 신고] 개인사업자 부가가치세 신고납부 : 일반과세자 신고내용 / 매입세액, 신용카드 사용내역 입력하기
  4. [홈택스 신고] 개인사업자 부가가치세 신고납부 : 일반과세자 신고내용 / 공제세액 / 예정고지세액 / 신고서 제출
